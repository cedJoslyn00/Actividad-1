### Consolidado de Requerimientos del Mercado ✔
*Basado en la investigación realizada, este es el perfil estándar solicitado:*

#### 💻 Tecnologías requeridas
- **Lenguajes de programación:** Python, R, SQL
- **Bases de datos y Data Warehouses:** MySQL, PostgreSQL, SQL Server, Snowflake, BigQuery
- **Herramientas de visualización:** Power BI, Tableau, Looker
- **Hojas de cálculo:** Microsoft Excel, Google Sheets
- **Herramientas adicionales:** Git, Jupyter Notebook, Databricks
- **Conocimientos básicos en:** Cloud (AWS, Azure o GCP)

#### 🛠 Habilidades técnicas (Hard Skills)
- Análisis y limpieza de datos (Data Cleaning)
- Consultas avanzadas en SQL
- Modelado de datos básico
- Creación de dashboards e informes interactivos
- Análisis estadístico básico
- Validación y control de calidad de datos
- Interpretación de tendencias y patrones

#### 🤝 Habilidades blandas (Soft Skills)
- Pensamiento analítico
- Atención al detalle
- Trabajo en equipo
- Capacidad de resolución de problemas
- Organización y gestión del tiempo
- Curiosidad y aprendizaje continuo

#### 📋 Responsabilidades principales
- Recopilar, analizar e interpretar datos para apoyar la toma de decisiones.
- Traducir preguntas de negocio en requerimientos analíticos.
- Crear y mantener dashboards y reportes periódicos.
- Identificar tendencias, patrones y oportunidades de mejora.
- Colaborar con equipos de negocio, ingeniería y tecnología.
- Documentar métricas, definiciones y procesos analíticos.
- Asegurar la calidad y consistencia de los datos.

### Patrones Identificados
- **Tecnología Común:** SQL es indispensable en el 100% de las ofertas. Python y una herramienta de BI (PowerBI/Tableau) aparecen en el 80%.
- **Habilidad Blanda Recurrente:** "Comunicación de insights" o "Storytelling con datos". No basta con analizar, hay que saber explicar.
- **Nivel de Inglés:** La mayoría solicita inglés intermedio/avanzado para documentación.

### Mi Perfil Objetivo
Analista de Datos enfocado en transformar datos crudos en decisiones de negocio estratégicas. Mi enfoque combina la precisión técnica (SQL/Python) con la narrativa visual (PowerBI/Tableau) para facilitar la toma de decisiones.

### Diferenciadores Clave
1. **Enfoque en Business Intelligence:** No solo entrego datos, entrego dashboards accionables para negocio.
2. **Portafolio Web Propio:** Demuestro habilidades técnicas desarrollando mi propia presencia web (HTML/CSS), mostrando versatilidad.
